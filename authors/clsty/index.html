    clsty

    Celestial.y

    clsty，实际上就是 Celestial.y 的缩写。读作 /sɪˈles.ti/ 。

    关于我

    兴趣爱好相当广泛，无法一言蔽之。

    你可以先简单地将我视作一位自由开源软件（Free and Open Source Software，FOSS）的爱好者。

    操作系统
    在 2023 年改用 Linux 并弃用 Windows 11。
    目前使用的发行版是 Arch Linux（桌面）和 Rocky Linux（服务器）。
    为了提高 archiso 的使用效率，开发了并维护着 arCNiso（GitHub 仓库）。
    桌面环境
    以 Hyprland + AGS 为核心的一套配置，分享于 dots-hyprarch。
    这套配置附带一个安装脚本，使得安装过程可交互且幂等（说人话就是稳定好用）。但目前仍未实现多语言，依然是纯英文。
    编辑器
    使用 Vim 的时间比使用 Linux 更长，不算精通但是很熟练。
    也会使用 Emacs，但重度依赖由 Evil 插件实现的 Vim 键位。
    Evil 有个实操型教程，即 evil-tutor 插件。我将其译为简体中文并完善功能，见 evil-tutor-sc。用此插件来入门 Vim 也没问题。

    本站提供 RSS 供读者订阅，一般每月更新一次，其中汇总了上个月站内的所有新内容（比如 4 月份的月摘汇总了 3 月份的内容）。1
